3. Función de activación Tanh¶

La funcin de activacin Tanh, viene determinada por la siguiente expresin matemtica:

$$\tanh(x) = \frac {\sinh(x)} {\cosh(x)}$$

Como puedes observar, para implementar la funcin Tanh, primero tienes que implementar las funciones:

Sinh:¶

$$\sinh(x) = \frac {e^{x} - e^{-x}} {2}$$

Cosh:¶

$$\cosh(x) = \frac {e^{x} + e^{-x}} {2}$$
Crea una funcin en Python que calcule el resultado de aplicar la funcin Tanh sobre una entrada x. Ten en cuenta que x debe ser considerado un valor numrico.

Pista: Cuando termines de programar la funcin, comprueba si el resultado es correcto utilizando la grfica que se muestra a continuacin.

def sinh(x):

def cosh(x):
    """Calcula el resultado de la funcion matematica cosh."""

    """Calcula el resultado de la funcion matematica tanh"""
